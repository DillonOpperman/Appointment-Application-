# IT Learning Center - Dependency Requirements

This document outlines all required dependencies for the IT Learning Center Appointment System.

## System Requirements

- Node.js v22.x or higher
- npm v10.x or higher
- MongoDB v5.x or higher (local or MongoDB Atlas)
- Git v2.x or higher

## Node.js Dependencies

### Production Dependencies

- express@^5.2.1 - Web application framework
- mongoose@^9.2.4 - MongoDB object modeling
- ejs@^5.0.1 - HTML templating engine
- bcryptjs@^3.0.3 - Password hashing
- jsonwebtoken@^9.0.3 - JWT authentication
- cookie-parser@^1.4.7 - Cookie parsing middleware
- csurf@^1.11.0 - CSRF protection middleware
- cors@^2.8.6 - Cross-origin resource sharing
- helmet@^8.1.0 - HTTP security headers
- express-rate-limit@^8.3.2 - Rate limiting middleware
- express-validator@^7.3.1 - Request validation middleware
- dotenv@^17.3.1 - Environment variable management
- nodemailer@^8.0.3 - Email sending
- googleapis@^171.4.0 - Google Calendar OAuth integration

### Development Dependencies

- nodemon@^3.1.14 - Auto-restart server on file changes
- jest@^30.3.0 - Testing framework
- supertest@^7.2.2 - HTTP integration testing

## Installation

```
npm install
```

## Running the App

```
npm start        # production
npm run dev      # development (auto-restart)
npm test         # run tests
```

## Environment Variables

Create a `.env` file in the project root with the following variables:

Required variables: MONGODB_URI, JWT_SECRET, EMAIL_USER, EMAIL_PASS, GOOGLE_CLIENT_ID, GOOGLE_CLIENT_SECRET, GOOGLE_REDIRECT_URI, NODE_ENV
